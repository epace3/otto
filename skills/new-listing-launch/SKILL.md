# New Listing Launch Workflow

## What this workflow does

Takes a new listing from "listing agreement signed" to "ready to go to market" in one guided pass. Produces 7 deliverables in sequence: intake confirmation, MLS description, feature sheet, launch-week social captions, seller intro email, photographer brief, and open house promo copy.

Everything is written in the agent's voice, using the listing-copy rules (lead with buyer emotional experience, features → benefits, no clichés, no AI tells).

## When to use

Trigger on:

- **"New listing — [address], here are my notes: [notes]"**
- "Run the new listing launch workflow"
- "Launch this listing: [address]"
- "I just signed a listing agreement on [address] — take it from here"
- Any variant where the agent is handing off a new listing for a full launch package

## Before you start

The workflow requires this intake data. If any is missing, **ask for it in one batch** before running Step 1:

1. **Property address**
2. **5 property notes** — what makes this place special (not specs, character)
3. **3 seller priorities** — fast sale / max price / minimal disruption / privacy / certainty
4. **List price**
5. **Target buyer profile** — who this property is written *for*

Do not start the workflow until you have all five. The copy only works if the intake is real.

---

## Step 1 — Intake Confirmation

Repeat back what you have in a short scannable format so the agent can confirm nothing is wrong before you start generating copy. Example:

> **Confirming intake for 742 Maple Drive:**
>
> - **List price:** $1,449,000
> - **Target buyer:** Move-up family, 35–45, trading a starter for space + Rosedale JHS catchment
> - **Seller priorities:** (1) close by end of June, (2) minimal disruption for the dog, (3) fair price — not reaching for a number the market won't support
> - **Property highlights:** 2023 kitchen reno; south-facing back garden with sun until 6pm; oversized garage the seller converted to a workshop; original 1940s hardwood on main floor; the street itself — the neighbours made it feel like a real community for the kids
>
> Everything accurate? I'll run the full launch package once you confirm.

Wait for confirmation or corrections before moving to Step 2.

---

## Step 2 — MLS Description

Write a **150–200 word** MLS description. Non-negotiable rules:

- **Lead with the buyer's emotional experience**, not the specs. First 1–2 sentences should put the target buyer *inside* the house.
- **Translate every feature into a benefit.** "4 bedrooms" → "a bedroom for each kid plus the guest room your mom actually looks forward to visiting." "Renovated kitchen" → "the Sunday-morning kitchen — the one where coffee turns into a two-hour conversation."
- **No clichés.** Banned: nestled, cozy, stunning, turnkey, must-see, charming, immaculate, pristine, gem, oasis, dream home, entertainer's delight, pride of ownership, meticulous, opportunity knocks, won't last.
- **No AI tells.** No "leverage," "utilize," "seamlessly," "foster," "enhance," etc.
- **End with a concrete next step** (open house date, private showings, a specific detail worth calling the agent about) — not "don't miss out."

**Present the MLS description to the agent and wait for feedback before moving to Step 3.** This is the one section where agent review before continuing is mandatory — the rest of the workflow builds on the copy direction you land here.

---

## Step 3 — Feature Sheet Copy

**Five bullet highlights.** Each one follows this structure:

**[Feature] — [why it matters to this specific buyer].**

Example:
- **South-facing back garden** — full sun until 6pm means summer dinners outside without chasing shade.
- **Converted garage workshop** — the dedicated space for the hobby, the project, or the home business that doesn't fit in a basement.
- **Rosedale JHS catchment** — the school district that holds resale value through the next market cycle.
- **Original 1940s hardwood, main floor** — character that can't be replicated by new builds at any price point.
- **2023 kitchen reno** — move-in ready, zero contractor coordination, zero hidden reno costs in year one.

Short. Punchy. Scannable. No prose.

---

## Step 4 — Social Caption Pack

**Five captions for launch week.** Vary the angle across the five so the agent can post one a day without feeling repetitive:

1. **Lifestyle angle** — put the reader inside a Sunday morning or a Friday dinner at this house
2. **Feature-forward angle** — the one thing about this house that stops people mid-scroll
3. **Neighbourhood angle** — what makes the *street* or the *pocket* worth buying into
4. **Urgency angle** — why the buyer who waits a week on this one loses (use sparingly, no fake urgency)
5. **Curiosity angle** — ask a question or lead with something that doesn't read like a listing at first

Rules:
- **Each caption standalone.** No "see previous post," no "part 1 of 5."
- **No hashtag spam.** Zero or 1–2 targeted hashtags only. Most captions don't need any.
- **Conversational.** Contractions. Short sentences. Specific details, not adjectives.
- **One strong hook in the first line** — that's what the buyer sees before they click "more."
- **A clear next step at the end** — open house date, DM for showings, link in bio, etc.

---

## Step 5 — Seller Intro Email

Written to the seller, from the agent. Covers:

- **What to expect** in the first week (photography, sign install, MLS launch, first showings)
- **Key dates** (photography day, go-live date, first open house)
- **What the agent needs from the seller**:
  - Decluttering / staging checklist (reference it, even if you don't produce it here)
  - Access for photography (confirmed day and window)
  - Sign install confirmation
  - Pet arrangement for showings if applicable
- **Warm close** — the seller should feel like they're in good hands, not like they just got a procedural email from a company

**Tone:** Warm, professional, clear. No filler, no "please don't hesitate," no "I hope this email finds you well." Match the agent's voice from their profile.

Include two subject line options at the top.

---

## Step 6 — Photographer Brief

Short, direct brief for the listing photographer. Covers:

- **Priority shots** — the 3–5 features worth capturing from the best possible angle (pulled from the intake notes)
- **Angles and times to consider** — e.g., "golden hour back garden shot" if the south-facing yard is a hero feature
- **Shots to avoid** — anything unflattering, mid-reno, or that undersells a hero room
- **Any specific details worth capturing** — a reading nook, a built-in, a view, a detail that only someone walking through would notice

Write it as a scannable brief, not a letter. The photographer should be able to glance at it once on site.

---

## Step 7 — Open House Promo Copy

Two versions:

**Long version — for social (one paragraph, ~60–80 words):**
Address, date *(placeholder if not confirmed)*, time *(placeholder)*, and the 1–2 features worth highlighting to pull traffic. Write it like a friend inviting another friend, not a sign on a lawn.

**Short version — for email (2–3 lines):**
The minimum the agent's database needs to know. Address, date, time, one hook. Clickable mental path: "oh, that sounds like my in-laws' kind of house" → forwarded to them.

---

## Final delivery

**Return all 7 outputs in one clean document**, with clear section headers for each step (`## MLS Description`, `## Feature Sheet`, etc.). The agent should be able to scroll the whole thing in one pass and pull each piece of copy into whatever tool they use to publish it.

**If this workflow is running inside a plugin environment (Otto Workspace, Cowork, Claude Code) with file-saving capabilities:** save each section as its own `.md` file in a folder named after the property (e.g., `742-maple-drive/`) and return clickable `computer://` links to each file at the end, so the agent can open each deliverable in the preview.

**Do not include commentary between sections.** The copy is the deliverable.

## Handoff note

After delivering all 7 outputs, close with **one line** telling the agent what comes next: schedule the photographer, confirm the go-live date, or start the `weekly-listing-update` workflow once the listing goes live. No paragraph — one line.
